universal-embed-player — live demo

Served as plain ES modules — no bundler required. Open this file with your Live Server extension. Every card below calls resolveSource() and createPlayer() directly from src/index.js. Check each card's "resolved source" panel and event log, plus your browser console, while testing.

Status: core resolvers + controller + native/HLS/DASH/iframe engines + interaction shield + Shadow-DOM control bar + thumbnail-first (light) mode are implemented and wired up here. Per plan.md §7/§8: native/HLS/DASH cards are genuinely brand-free (no iframe, no vendor chrome); iframe cards (YouTube/Vimeo/Wistia) are "brand-minimized," not "brand-free" — their own postMessage protocol drives custom controls only where implemented (YouTube, Vimeo).
